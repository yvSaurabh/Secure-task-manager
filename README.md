 Secure Task Manager

A secure full-stack task management application built using Node.js, Express, MongoDB, and Vanilla JavaScript.

 Live Demo
https://secure-task-manager-nnu9.onrender.com

 Tech Stack
- Node.js
- Express.js
- MongoDB
- Vanilla HTML, CSS, JavaScript
- JWT Authentication
- AES Encryption

 Features
- User Registration & Login
- JWT Authentication (HTTP-only cookies)
- Protected Routes
- Task CRUD (Create, Read, Update, Delete)
- Pagination & Search
- Task Status Update (Mark Completed)
- Secure Logout
- AES-encrypted task descriptions

Security Implementations
- Password hashing using bcrypt
- JWT stored in HTTP-only cookies
- Authorization middleware
- User-based data isolation
- AES encryption for sensitive fields
- Environment variables for secrets
